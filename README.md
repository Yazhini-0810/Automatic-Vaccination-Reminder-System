# Automatic Vaccination Reminder System

## Overview
The Automatic Vaccination Reminder System is a Python-based application that helps parents remember their children's vaccination schedules by sending automated email reminders. The system stores vaccination details, generates reminders at the scheduled time, and provides educational information about the vaccine.

## Features
- Register child vaccination details
- Automated email reminders using SMTP
- Vaccine-specific educational information
- User-friendly command-line interface
- Supports future SMS notification integration

## Technologies Used
- Python
- SMTP (Simple Mail Transfer Protocol)
- MIME (Email Formatting)
- Datetime
- Time Module

## Project Workflow
1. User enters child and vaccination details.
2. The system stores the information.
3. The application waits until the scheduled reminder time.
4. An automated email reminder is sent to the parent's email.
5. Educational information about the vaccine is included in the email.

## Future Enhancements
